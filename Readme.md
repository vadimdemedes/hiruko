Development server for single-page apps.
Renders `index.html` if a request URL does not have an extension.


## Installation

```
$ npm install --global hiruko
```


## Usage

Serve `public` directory:

```
$ hiruko public
```

Serve current directory:

```
$ hiruko
```

Use port `8080` for a server:

```
$ PORT=8080 hiruko
```
